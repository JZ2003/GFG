# 35L project

# Build Guideline

### Before all, use Homebrew to install Node if you don't have one

`brew install node`

### Clone this repo

`git clone https://github.com/TraderBros-UCLA/ucla-22f-35l-project.git`

### Go to your cloned directory, install React if first time building

`cd frontend`

`npm install`

### Run App

`npm start`
